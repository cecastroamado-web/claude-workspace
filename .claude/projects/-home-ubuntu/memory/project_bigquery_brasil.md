Integrar dados do Brasil (BigQuery) ao dashboard financeiro internacional para cruzar:
- Resultado operacional Brasil vs. internacional
- Gasto real de mídia Brasil (Facebook/Google) vs. repasses que os países internacionais enviam para Reweb Corp
- Gap: quanto Brasil precisa enviar para cobrir a mídia que o resultado internacional não cobre

**Why:** Hoje o dashboard mostra os 6 países internacionais mas não tem os dados do Brasil. Sem o Brasil, não dá para fechar o quadro completo de fluxo de mídia e resultado consolidado.

**How to apply:** Quando o usuário quiser avançar nessa integração, o primeiro passo é conectar o BigQuery (projeto Google Cloud) e mapear as tabelas relevantes (receita, despesa, mídia Brasil).
